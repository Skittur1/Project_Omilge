# Omegle Clone

A real-time anonymous chat application inspired by Omegle, allowing users to connect with random strangers for one-on-one conversations.

## Features

* Random user matching
* Real-time messaging
* Anonymous chat experience
* User connection and disconnection handling
* Secure communication

## Tech Stack

* WebRTC
* ASP.NET Core Web API
* SignalR
* TSQL

### Database

* SQL Server

## Getting Started

### Prerequisites

* .NET SDK
* SQL Server
* SSMS
* Visual Studio
